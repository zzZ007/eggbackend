#### 软件架构

框架：``EggJS``

ORM框架：``sequelize``

模板引擎：``nunjucks``

数据库：``Mysql``

后端框架：``AdminLTE``

#### 安装教程

1.  安装依赖: ```yarn install / npm install```
2.  导入数据库: ``` db/db.sql ```
3. 测试账号: ```admin/123456```

#### 使用说明

开发环境启动： ```npm run dev```

正式环境启动： ```npm run start```

前端地址：``` 127.0.0.1:8080 ```

后台地址：``` 127.0.0.1:8080/admin ```
